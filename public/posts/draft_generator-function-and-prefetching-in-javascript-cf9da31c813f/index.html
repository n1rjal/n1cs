Generator function and prefetching in javascript

There are many JS tricks that are unknown to many people. Here are some of the examples that I have been using in my projects.

Generator Functions:

Generator functions in JavaScript are a special kind of function that can be paused and resumed, which allows you to write code that can be executed in a stop-and-start manner.

To define a generator function, you use the function* syntax. Inside a generator function, you use the yield keyword to pause the function. When the generatorâ€™s next() method is called, the function resumes execution until it reaches the next `yield` statement.

Here is an example of a simple generator function:

function* simpleGenerator() {
yield 1;
yield 2;
yield 3;
}
const gen = simpleGenerator();
console.log(gen.next()); // {value: 1, done: false}
console.log(gen.next()); // {value: 2, done: false}
console.log(gen.next()); // {value: 3, done: false}
console.log(gen.next()); // {value: undefined, done: true}

In the example above, simpleGenerator is a generator function that yields the values 1, 2, and 3. When we call simpleGenerator(), we get a generator object (`gen`), which has a next() method. Each call to next() resumes the generator function until it reaches the next yield, and returns an object with the yielded value and a boolean indicating whether the generator is done.

Generator functions can also receive input through the next() method:

function* echoGenerator() {
while(true) {
const input = yield;
console.log(input);
}
}
const echo = echoGenerator();
echo.next(); // Starts the generator
echo.next(‘Hello, World!’); // Logs: Hello, World!
echo.next(‘Generators are cool!’); // Logs: Generators are cool!

In this example, `echoGenerator` is an infinite generator function that logs whatever input it receives. We first need to call `next()` to start the generator, and then we can pass values using `next()`.

Generator functions are a powerful feature of JavaScript that allow you to write asynchronous code in a synchronous style, among other things.
